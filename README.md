# 使用前注意事项

## 自动评价

    by Eric
    本脚本的目的是避免点点点的反人类操作，以及对老师工作的肯定，因此默认全部好评，
但是对老师的文字评价需要自己书写。如果某门课不想全部好评,请按“q”跳过，然后自行进入系统进行评价！
### 注意
有些课可能还没有开始上，因此暂时先不用评价！跳过就好！切记！
另外建议时间充足的同学自行进入系统手动评价！给老师评价真实成绩，有利于老师改进教课方法，最后受益者是自己！
